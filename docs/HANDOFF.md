幫美國客戶 **Thunk Works LLC** 開 invoice 與統一發票（三款包裝設計專案）。

## 關鍵結論（拍板）

- **客戶**：Thunk Works LLC
  - 地址：122A E Foothill Blvd #104, Arcadia, CA 91006（查過是 UPS Store 私人信箱，不是實體辦公室）
  - 該公司在台灣**沒有分公司、沒有台灣帳戶、沒有統編**

- **付款條件**：Net 30
- **稅務處理**：申請**外銷勞務零稅率（0%）**，省 5% 營業稅（約 1,500 元）

## 為什麼選這個方案
- 客戶要用 LLC 報帳 → **必須**用 LLC 帳戶付款（不能用個人）
- LLC 沒有台灣帳戶 → 只能從美國匯美金
- 美金匯款有外匯水單 → 可申請**外銷勞務零稅率**（省稅 1,500 元）抵銷銀行手續費

## 其他備選方案（已排除）
- ~~客戶個人台幣匯款~~：會造成帳實不符（invoice 開 LLC、匯款人是個人），他 LLC 報不了帳
- ~~台幣一次付 / 分兩次~~：客戶 LLC 沒台幣帳戶，走不了
- ~~含稅 31,500~~：走美金零稅率就不用加 5% 稅

### 2. 自己準備好
- [ ] 公司英文名稱 + 英文地址
- [ ] **美金收款帳戶資訊**（向富邦外匯部門索取）：
  - 銀行英文名
  - SWIFT Code
  - 帳號
  - 戶名（英文）

- [ ] 三款包裝的英文品項寫法（可以合成一行 "Packaging design for 3 SKUs" 或分三行）

### 3. 確認會計師
- 零稅率申報自己做容易出錯，建議**找會計師處理**
- 會計師要知道：收美金、有外匯水單、外銷勞務零稅率

### 4. 收到款後的流程
1. 客戶匯美金 → 進富邦美金戶頭
2. 向富邦**索取外匯水單**（要留存，會計師要用）
3. 查當日匯率換算台幣
4. 開**統一發票**：
   - 稅率類別選「**外銷零稅率**」（**不是「免稅」**）
   - 銷售額 = 水單換算後的台幣金額（例 TWD 30,846）
   - 稅額 = 0
5. 把水單 + 發票存檔，交給會計師報營業稅

## 提醒（下次別踩坑）

1. **統一發票要選「零稅率」不是「免稅」**
   - 零稅率可以扣抵進項稅
   - 免稅不行
   - 兩個系統都有選項，別按錯

2. **外匯水單一定要跟富邦拿**
   - 沒水單 → 申請不了零稅率 → 白吃 5% 稅

3. **發票開立時機**
   - 收到款**當期**（當月）開，不是動工時

4. **客戶那個地址是 UPS Store 信箱**
   - 這對請款沒影響（正常商業慣例）
   - 但要知道他不是什麼大公司實體辦公室
